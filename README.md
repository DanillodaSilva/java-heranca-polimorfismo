# Java POO - Herança e Polimorfismo

Projeto desenvolvido para praticar conceitos de Programação Orientada a Objetos em Java.

## Conceitos praticados

- Encapsulamento
- Herança
- Polimorfismo
- Classes abstratas
- Métodos sobrescritos
- Organização de código

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git/GitHub

## Objetivo do projeto

O objetivo deste projeto foi praticar conceitos fundamentais de POO através da criação de diferentes tipos de usuários e comportamentos utilizando herança e polimorfismo.

## Como executar

1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`
